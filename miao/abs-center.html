CSS 实现子元素在父元素垂直和水平居中

说明:父元素和子元素宽度固定, 高度可变

垂直居中

方案 1: 定位(子元素过分受限配合 margin:atuo)

  1. 父元素 position: relative; 子元素 position: absolute
  2. 子元素设置高度
  3. 子元素过分受限: top:0;bottom:0;
  4. 子元素设置 margin-top: auto;margin-bottom: auto;

方案 2: 定位(子元素过分受限配合 margin:atuo)

  1. 父元素position: relative; 子元素 position: absolute
  2. 子元素设置 top: 父元素高度的 50% - 子元素高度的50%

方案 3: 行内布局

  1. 子元素设为行内元素
  2. 子元素 line-height 等于父元素 content-area 高度(父元素内的匿名文本垂直居中)
  3. 子元素 vertical-align: middle (设置子元素行内框的中间位置对齐匿名文本中 x 字符 baseline 向上 0.5ex 的位置)

方案 4: flex 布局

  1. 父元素设置 flex 布局: display: flex;
  2. 方法 1--利用主轴方向居中
    1. 父元素设置 flex 布局主轴方向为垂直: flex-direction: column;
    2. 父元素设置主轴方向内容居中: justify-content: center;
    方法 2--利用从轴方向居中
    1. 父元素设置主轴方向内容居中: align-items: center;

水平居中

方案 1:定位(子元素过分受限配合 margin:atuo)

  1. 定位: 父元素 position: relative;子元素 postion: absolute;
  2. 子元素过分受限: top:0;bottom:0;left:0;right:0;
  3. 子元素自动外边距: margin-top: auto; margin-bottom:auto;

方案 2:定位(设置子元素 top)

  • 方法 1:子元素 margin-left 设置为(父元素 content-area 宽度 - 子元素宽度) * 50%
  • 方法 2:子元素 margin-left 设置为 父元素 content-area * 50%, 同时 transform: translateX(-50%);

方案 3:行内布局

  1. 子元素设为行内元素
  2. 父元素行内文字水平居中 text-align: center;

方案 4:flex 布局

参考垂直居中方案可做
